# 個人網站模板

這是一個受 [Andrej Karpathy 的網站](https://karpathy.ai/) 啟發的簡單個人網站模板，專為 GitHub Pages 設計。

## 功能

- 簡潔、響應式設計
- 純 HTML 和 CSS（無框架）
- 包含經歷、教育、項目、寫作等部分
- 易於自訂

## GitHub Pages 設定

1. Fork 或複製此倉庫。
2. 編輯 `index.html` 以個人化您的資訊。
3. 將 `assets/profile.jpg` 替換為您的照片。
4. 前往您的倉庫設定 > Pages > Source: Deploy from a branch > Branch: main > 儲存。
5. 您的網站將在 `https://yourusername.github.io/repository-name/` 上線。

## 自訂

- 更新 `index.html` 中的內容。
- 修改 `style.css` 中的樣式。
- 根據需要新增更多部分或圖片。
